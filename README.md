# PetroGM-test-task

## Описание
Консольное приложение, задание.

## Требования
- C++ 20 или выше
- CMake 3.27 или выше

## Сборка
```bash
cmake -B build
cmake --build build
```
## Запуск (тест)
```bash
./TEST.sh
```

## Клонирование -> Сборка -> Запуск
```bash
git clone https://github.com/Plush-Jill/PetroGM-test-task.git
cd PetroGM-test-task
cmake -B build
cmake --build build
chmod +x ./TEST.sh
./TEST.sh
```
